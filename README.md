# 💻 C Programming Practice Repository

A collection of fundamental and intermediate-level **C programming problems** designed to strengthen problem-solving skills and core programming concepts.

---

## 📌 Overview

This repository contains multiple C programs covering essential topics such as:

- Basic number operations
- Mathematical computations
- Pattern printing
- String manipulation
- Logical problem solving

It is intended for **learning, practice, and revision of C programming concepts**.

---

## 📂 Project Structure

The repository includes the following types of programs:

### 🔢 Number-Based Programs
- Factorial of a number
- Greatest Common Divisor (GCD)
- Power of a number
- Reverse a number
- Count digits in a number
- Sum of first _n_ natural numbers
- Prime number check
- Prime numbers within a range
- Decimal to binary conversion
- Largest element in a set

---

### 🔤 String Program
- String reversal

---

### 🔺 Pattern Programs
- Star Pattern 1 to 8
- General pattern printing

---

### ➗ Other Programs
- Multiplication operations
- Miscellaneous logic-building problems

---

## 🛠️ Tech Stack

- **Language:** C
- **Compiler:** GCC / Turbo C / Any standard C compiler
- **Platform:** Cross-platform (source code only)
